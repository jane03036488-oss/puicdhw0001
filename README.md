# CUDA GPU Vector Addition (C 語言 + CUDA 高效能計算)

本專案示範如何使用 **C 語言搭配 NVIDIA CUDA** 在 GPU 上進行平行運算，以向量加法（Vector Addition）作為高效能計算的入門範例。  
此程式比較 CPU 與 GPU 在大量資料運算時的效能差異，讓初學者能了解 GPU 平行運算的優勢。

---

## 🔧 功能說明

- 使用 C 語言撰寫 CUDA Kernel
- 使用 GPU 平行化計算向量加法：C = A + B
- 展示 GPU 運算流程：記憶體配置、資料傳輸、Kernel 執行
- 能執行 100 萬筆資料以上的加法運算
- 具有良好的可讀性，適合課堂專題與大一學生理解
